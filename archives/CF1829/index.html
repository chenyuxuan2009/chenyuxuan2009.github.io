A.Love Story

题目翻译

给出一个长度为 101010 的字符串，一位一位看问它与 codeforces 差了多少字符。

ttt 组数据，1≤t≤10001\leq t\leq 10001≤t≤1000。

题目思路

按照题意模拟，扫描一遍给出字符串，判断第 iii 位与 codeforces 的第 iii 位是否相同即可。

完整代码

#include<bits/stdc++.h>
using namespace std;
const string t="codeforces";
void solve()
{
    string s;
    int ans=0;
    cin>>s;
    for(int i=0;i<10;i++)
    {
        if(s[i]!=t[i])
        {
            ans++;
        }
    }
    cout<<ans<<endl;
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

B.Blank Space

题目翻译

给出一个长度为 nnn 的序列 aaa，问最长连续 000 有多长。

1≤n≤100,0≤ai≤11\leq n\leq 100,0\leq a_i\leq 11≤n≤100,0≤ai​≤1。

ttt 组数据，1≤t≤10001\leq t\leq 10001≤t≤1000。

题目思路

我们遇到一个 000，便把长度加上 111。如果当前位不是 000，说明已经有了一段连续 000 了，计算答案 ans=max⁡{ans,len}ans=\max\{ans,len\}ans=max{ans,len}。

注意循环结束也要计算一遍，防止最后一个是 000 的情况不会统计长度。

完整代码

#include<bits/stdc++.h>
using namespace std;
void solve()
{
    int n;
    cin>>n;
    int len=0,ans=0;
    for(int i=1;i<=n;i++)
    {
        int x;
        cin>>x;
        if(x!=0)
        {
            ans=max(ans,len);
            len=0;
        }
        else
        {
            len++;
        }
    }
    ans=max(ans,len);
    cout<<ans<<endl;
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

C.Mr. Perfectly Fine

题目翻译

有 nnn 本书，每本书耗时 aia_iai​。有一个长度为 222 的字符串 skillskillskill，skilljskill_jskillj​ 为 1/0\tt{1/0}1/0 时表示是否获得了第 jjj 种技能。问最少耗时多久才能学习所有技能。无解输出 −1-1−1。

n≤2×105,1≤ai≤2×105n\leq 2\times 10^5,1\leq a_i\leq 2\times 10^5n≤2×105,1≤ai​≤2×105。

ttt 组数据，1≤t≤1000,∑tt=1tn≤2×1051\leq t \leq 1000,\sum_{tt=1}^t n\leq 2\times 10^51≤t≤1000,∑tt=1t​n≤2×105。

题目思路

我们分两种情况讨论，第一种是看两本书学习两个技能，第二种是看一本书学习所有技能。对于两种情况，答案取最小值。

完整代码

#include<bits/stdc++.h>
using namespace std;
void solve()
{
    int n;
    cin>>n;
    int m1,m2,m12;
    m1=m2=m12=(1<<29);
    while(n--)
    {
        int x;
        string s;
        cin>>x>>s;
        if(s=="11")
        {
            m12=min(m12,x);
        }
        else if(s=="10")
        {
            m1=min(m1,x);
        }
        else if(s=="01")
        {
            m2=min(m2,x);
        }
    }
    cout<<(min(m1+m2,m12)==(1<<29)?-1:min(m1+m2,m12))<<endl;
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

D.Gold Rush

题目翻译

有一堆含有 nnn 个石子的石子堆，你每次可以选择大小为 kkk 的一堆，将其分成 k3\frac{k}{3}3k​ 与 2×k3\frac{2\times k}{3}32×k​ 两堆，问是否可以得到大小为 mmm 的堆。

1≤n,m≤1071\leq n,m\leq 10^71≤n,m≤107。

ttt 组数据，t≤1000t\leq 1000t≤1000。

题目思路

我们通过阅读题面，因为这个 nnn 是要分成三份然后一堆一份一堆两份，显然可以得到 n,mn,mn,m 只有可以表示为 n=3i×p,m=2j×p(0≤j≤i,p≥0)n=3^i\times p,m=2^j\times p(0\leq j\leq i,p\geq 0)n=3i×p,m=2j×p(0≤j≤i,p≥0) 时，我们才可以分出来。

考虑到 n,mn,mn,m 都很小，我们枚举这个 i,ji,ji,j，判断是否可行即可。

完整代码

#include<bits/stdc++.h>
using namespace std;
void solve()
{
    int n,m,tmp3=1,tmp2;
    cin>>n>>m;
    for(int i=0;i<=15;i++)
    {
        tmp2=1;
        for(int j=0;j<=i;j++)
        {
            if(n%tmp3==0&&n/tmp3*tmp2==m)
            {
                puts("YES");
                return;
            }
            tmp2*=2;
        }
        tmp3*=3;
    }
    puts("NO");
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

E.The Lakes

题目翻译

你有一个 n×mn\times mn×m 的矩阵，其中有水坑，水坑的大小为 ai,ja_{i,j}ai,j​。若 ai,j=0a_{i,j}=0ai,j​=0 则不是水坑。问最大的连续水坑大小是多少。

1≤n,m≤1000,0≤ai,j≤10001\leq n,m\leq 1000,0\leq a_{i,j}\leq 10001≤n,m≤1000,0≤ai,j​≤1000。

ttt 组数据，1≤t≤1000,∑tt=1tn×m≤1061\leq t \leq 1000,\sum_{tt=1}^t n\times m\leq 10^61≤t≤1000,∑tt=1t​n×m≤106。

题目思路

典型的四方向连通块，使用搜索，深搜广搜均可。

记录每个水坑是否走过避免回头路。

注意开始遍历的时候要判断当前位置是否为 000。

1
2 2
0 1
1 0

如果不判断的话会从 (1,1)(1,1)(1,1) 开始找到 (1,2)(1,2)(1,2) 和 (2,1)(2,1)(2,1)，但其实这两个点不连通。

完整代码

#include<bits/stdc++.h>
using namespace std;
int n,m;
bool vis[1020][1020];
int a[1020][1020];
int dx[]={0,0,-1,1};
int dy[]={-1,1,0,0};
bool judge(int x,int y)
{
    return x>=1&&y>=1&&x<=n&&y<=m&&!vis[x][y]&&a[x][y]>0;
}
int dfs(int x,int y)
{
    vis[x][y]=1;
    int ans=a[x][y];
    for(int i=0;i<4;i++)
    {
        if(judge(x+dx[i],y+dy[i]))
        {
            ans+=dfs(x+dx[i],y+dy[i]);
        }
    }
    return ans;
}
void solve()
{
    cin>>n>>m;
    memset(vis,0,sizeof(vis));
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            cin>>a[i][j];
        }
    }
    int ans=0;
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            if(!vis[i][j]&&a[i][j])
            {
                ans=max(ans,dfs(i,j));
            }
        }
    }
    cout<<ans<<endl;
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

F.Forever Winter

题目翻译

我们定义大小 (x,y)(x,y)(x,y) 的雪花图表示有一个节点 uuu 连向 xxx 个节点，这 xxx 个节点又分别连向不同于 uuu 的 yyy 个节点。给定一张 nnn 节点 mmm 条边的雪花图，问这张雪花图的大小是什么。

例如 就是大小为 (5,3)(5,3)(5,3) 的雪花图。

1≤n≤200,1≤m≤min⁡(n×(n−1)2,1000)1\leq n\leq 200,1\leq m\leq \min(\frac{n\times (n-1)}{2},1000)1≤n≤200,1≤m≤min(2n×(n−1)​,1000)。

题目思路

因为给出的图一定是雪花图，所以我们只需枚举它的大小。然后我们观察样例可得，我们需要有一个度数为 xxx 的点以及 xxx 个度数为 y+1y+1y+1 的点（因为连到了 xxx 个点，每个点还会往中心连一次）。

但仅仅这些还不够，我们需要判断这个 (x,y)(x,y)(x,y) 是否满足有 nnn 个点，即 x×y+1=nx\times y+1=nx×y+1=n。

完整代码

#include<bits/stdc++.h>
using namespace std;
int deg[220];
void solve()
{
    memset(deg,0,sizeof(deg));
    int n,m;
    cin>>n>>m;
    while(m--)
    {
        int u,v;
        cin>>u>>v;
        deg[u]++;deg[v]++;
    }
    for(int x=1;x<=200;x++)
    {
        for(int y=1;y<=200;y++)
        {
            if(x*(y+1)+1!=n)continue;
            int sx=0,sy1=0;
            for(int i=1;i<=n;i++)
            {
                if(deg[i]==x&&!sx){sx=1;continue;}
                if(deg[i]==y+1){sy1++;}
            }
            if(sx==1&&sy1==x)
            {
                cout<<x<<" "<<y<<endl;
                return;
            }
        }
    }
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

G.Hits Different

题目翻译

你有一个罐子搭成的金字塔，如图。第 iii 个罐子代表数 i2i^2i2，现在打掉第 iii 个罐子，问掉落罐子的数之和为多少。

1≤n≤1061\leq n\leq 10^61≤n≤106。

ttt 组数据，1≤t≤10001\leq t\leq 10001≤t≤1000。

题目思路

观察可得，打掉第 xxx 行第 yyy 列的罐子，那么第 xxx 行第 y−1y-1y−1 和第 xxx 行第 yyy 列的罐子都会倒塌，然后逐个往上。

我们做一个预处理，只需找出 nnn 罐子的位置，一个个往上递推，用前缀和优化记录某一行某一段的和即可。

完整代码

#include<bits/stdc++.h>
using namespace std;
int x,l,r;
long long a[2020][2020],cnt;
long long s[2020][2020];
pair<int,int>fa[3000020];
long long Sum(int i,int l,int r)
{
    if(l<1)l=1;
    if(r>i)r=i;
    return s[i][r]-s[i][l-1];
}
void Get(int n)
{
    x=fa[n].first;
    l=r=fa[n].second;
}
void solve()
{
    int n;
    long long ans=0;
    cin>>n;
    Get(n);
    while(x>0)
    {
        ans+=Sum(x,l,r);
        l--;
        x--;
    }
    cout<<ans<<endl;
}
int main()
{
    for(int i=1;i<=2000;i++)
    {
        for(int j=1;j<=i;j++)
        {
            a[i][j]=++cnt;
            s[i][j]=s[i][j-1]+a[i][j]*a[i][j];
            fa[cnt]=make_pair(i,j);
        }
    }
    int t;cin>>t;while(t--)
    solve();
    return 0;
}

H.Don't Blame Me

题目翻译

你有一个长度为 nnn 的数组 aaa，问有多少子序列使得选中数字按位与结果中含有 kkk 个 111，答案对 109+710^9+7109+7 取模。

1≤n≤2×105,0≤k≤6,0≤ai≤631\leq n\leq 2\times 10^5,0\leq k\leq 6,0\leq a_i\leq 631≤n≤2×105,0≤k≤6,0≤ai​≤63。

ttt 组数据，1≤t≤1041\leq t\leq 10^41≤t≤104。

题目思路

这题非常像背包，但是背包的转移是 f[j]=max(f[j],f[j+w[i]]+value[i]);，这题只不过是把所谓体积的转移从加法改成按位与。

这题 dp 的初值也很有意思，赛时因为最开始的初值赋错导致痛失 AK。

初值应该写成 f[63]=1，假设先放上一个 (111111)2(111111)_2(111111)2​ 去做按位与。因为 000 按位与答案一定是 000，后面就没意义了。

但是这样有个问题，当 k=6k=6k=6 因为加了一个额外的 (111111)2(111111)_2(111111)2​，所以会选择空集。k=6k=6k=6 我们最后答案减一即可。

完整代码

#include<bits/stdc++.h>
using namespace std;
const int p=1000000007;
void solve()
{
    int n,m;
    long long ANS=0;
    cin>>n>>m;
    int a[n+1];
    long long f[64]={},ans[64]={};
    for(int i=1;i<=n;i++)
    {
        cin>>a[i];
    }
    ans[63]=1;
    for(int i=1;i<=n;i++)
    {
        for(int j=0;j<64;j++)
        {
            f[a[i]&j]=(f[a[i]&j]+ans[j])%p;
        }
        for(int j=0;j<64;j++)
        {
            ans[j]+=f[j];
            f[j]=0;
        }
    }  
    for(int i=0;i<64;i++)
    {
        if(__builtin_popcount(i)==m)
        {
            ANS=(ANS+ans[i])%p;
        }
    }
    cout<<ANS-(m==6)<<endl;
}
int main()
{
    int t;cin>>t;while(t--)
    solve();
    return 0;
}
