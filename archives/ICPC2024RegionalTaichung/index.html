M. Selection Sort

给定长度为 nnn 的序列 aaa，对长度为 mmm 的序列使用一次 Alice-Sort 的代价为 m2m^2m2。

int alice_sort(int *s, int n){
  for(int i = 0; i < n; ++i){
    for(int j = i + 1; j < n; ++j){
      if(s[i] > s[j]){
        int swap = s[i];
        s[i] = s[j];
        s[j] = swap;
      }
    }
  }
  return 0;
}

有两种操作：

可以使用 Alice-Sort 对一个前缀排序。

可以使用 Alice-Sort 对一个后缀排序。

每种操作限用一次。
