变量定义

一般是按题目给定。

保存 .size() 用 $sz$

数组大小一般只有 $n,m,N,M$。部分时候也用 $sz$。

数组无特殊就是 $a,b,c$ 之类的。

计数或答案用 $s,sum,c,cnt,ans,z$ 等。

多组数据肯定是 $t,T$。

树的边输入用 $x,y$。然后 a[x].push_back(y) 之类的。

存边用 $a$ 或 $tree$。

打标记 $f,ff,f1,f2$。

最大最小 $maxx,minn$。
