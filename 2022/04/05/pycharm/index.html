下载软件

首先进入官网，https://www.jetbrains.com/pycharm，进去之后，点击黑色的 DOWNLOAD。

点击之后，进入这样的页面。


一定选择右边的，右边的是社区版，虽然缺失部分功能，却是免费的。

左边是正版，有钱的话，还是支持一下，付点钱，但是有点小贵。

下载好之后，在下载里面有这样图案。

下载的东西一般都在 此电脑\下载 或 此电脑\Download。

安装软件

双击点开，这样的页面，点击 Next。

这个是设置存放的目录，看个人喜好，我放在 D盘。

点击 Next。

全都打上勾，Next。

不用管，直接 Install。

这时候就开始下载了。

下载好之后，直接重启电脑，选择第一个，打钩，Finish。

下载好后，电脑出现这样的图标，就成功了。

打开是这样的页面。

新建一个 Project。

我这边是选择在桌面上建立一个名为 Python 的文件夹，放代码。改第一行就好，其他不动，Create。

打开这个样子。

配置软件

设置主题

我们不设置东西，打开是纯黑色界面，看得难受。

所以我我们来更改颜色主题。

左上角，File-Settings。

找到 Editor-Color Scheme。

直接更改，我选择的是 Github 主题，点击 OK。

此时，颜色调整好了。

设置中文

老地方，File-Settings。

打开后，找到 Plugins。

搜索 Chinese。

无法查找？提示刷新网络依旧不出现插件页面？点击左侧小三角，展开，进行讲解。

出现这种情况：
1.打开 C盘-Windows-System32-drivers-etc-hosts。
2.进入 ipaddress，搜索 plugins.jetbrains 的ip。
3.找到第一个 ip，进入 hosts 文件，黏贴 查找到的ip plugins.jetbrains.com 这样一段字符。比如我的是 108.156.107.47 plugins.jetbrains.com。这个是不一定的，ip 地址是自己搜索的结果。
4.回 Pycharm，刷新试试。

选择第几个是不重要的，有时候可能会变。但里面一定要出现 Simplified 的字眼。

点击下载。

下载好后，点击 Restart IDE，重启 Pycharm。

再次打开就变成中文了。

配置解释

有的时候，特别是 Python 爬虫编写，会报错。我们就要下载这些东西。

设置-项目:Python-Python 解释器。

插句嘴，像 BeautifulSoup4 这样非自带的，本地下载是打开 cmd，输入以下内容。

pip3 install BeautifulSoup

其他同理。

话说回来，Python 解释器，双击 pip。

打开。

搜索 BeautifulSoup。

下载。

其他会用到的，都可以下载。

这样，Pycharm 就差不多配置完了。有其他的，欢迎评论，我一一填坑。
